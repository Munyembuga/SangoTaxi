# Sango Taxi

A Flutter-based ride-hailing application.

## Features

- **Ride Booking**: Book rides with nearby drivers
- **Real-time Tracking**: Track your ride in real-time  
- **Multiple Payment Options**: Pay securely through various methods
- **Profile Management**: Manage your account and preferences
- **Guest Mode**: Use the app without creating an account
- **Multi-language Support**: Available in multiple languages
